# About And Contact

Sources:

- `src/pages/about.astro`
- `src/pages/contact.astro`

## About Page

The about page explains RicoFast as a template project, not a backend SaaS company. It covers:

- Why the template exists.
- Who it is for.
- Values and product philosophy.
- Project timeline/community context.
- Links to GitHub or relevant project resources.

## Contact Page

The contact page provides a polished static contact flow. It includes:

- Contact page header.
- Demo form fields.
- Subject/topic options.
- Local success state.
- Contact/support context.

The form is a template UI. It does not submit to a real backend by default.
