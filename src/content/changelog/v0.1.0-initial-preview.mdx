## What's included

- Core marketing pages for Home, Features, Pricing, Blog, Changelog, About, Contact, Elements, and 404
- A reusable section system for hero areas, feature blocks, pricing cards, comparison tables, FAQ, tech stack, use cases, and final CTAs
- MDX-powered blog with featured posts, pagination, tags, reading progress, and table of contents
- Changelog timeline with version badges and MDX content support
- Built-in dark mode with no flash on load
- Responsive layouts across phone, tablet, laptop, and desktop breakpoints
- AOS scroll reveals with small custom presets
- Animated hero text entrance with staggered timing
- Pricing toggle for monthly and yearly billing examples
- Accordion items for FAQ sections
- Badge component with primary, accent, success, info, warning, and danger variants
- SEO-ready metadata, Open Graph tags, sitemap, and RSS feed
- Elements reference page for checking tokens, components, and UI states

## Why it stayed pre-1.0

This release proved the shape of the template, but the content model and visual story still needed refinement. We kept it in the `v0.x` line so teams could test the foundation without treating the public API as final.
